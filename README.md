# My Calculator Project!

This is the repo for my calculator project

## Learning Outcomes

Through this project, I have learned:
* Creating a calculator from scratch using just Javascript, HTML and CSS
* Using CSS flexbox to position elements on the page
* Using Javascript to dynamically update UI elements on the page
* Added keyboard support to interact with the calculator

## Link to Live Project

https://bagasprimanto.github.io/the-calculator/

## How to use

Use the calculator like you would an actual calculator!
Additionally, try pressing buttons on your keyboard to interact with the calculator!
